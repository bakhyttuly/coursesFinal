# Courses Final Project

##  Description
This is a Spring Boot web application for managing online courses.
The system supports user registration, authentication, course creation,
lessons, and student enrollment.

##  Tech Stack
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Liquibase
- Thymeleaf
- MapStruct
- Docker
- JUnit 5 + Mockito
